Request for Comments (RFCs)
===========================

What is a RFC?
--------------

RFCs were the main mechanism used in the early days of the Avocado
project to discuss larger ideas.  RFCs have been pretty much
supplanted by BluePrints (see :doc:`/blueprints/BP000`).
